# Поиск мебели по фото — MVP

Прототип визуального поиска товаров по фотографии. Модель CLIP работает прямо
в браузере, без бэкенда. Загружаешь фото интерьера — получаешь похожие товары
из каталога с автоопределением категории и фильтрами.

Демо-версия: каталог и изображения обезличены, бренд и внешние ссылки удалены.

## Структура

```
visual-search-demo/
├── index.html       ← страница с поиском (вся логика тут)
├── catalog.json     ← каталог: 234 товара, векторы CLIP, теги
├── images/          ← локальные фото товаров (0000.jpg …)
├── demo-photos/     ← примеры фото для проверки поиска
├── fonts/           ← шрифт интерфейса
└── README.md
```

## Как запустить

```bash
cd visual-search-demo
python3 -m http.server 8080
```

Открой http://localhost:8080

При первом запуске браузер скачает модель CLIP (~50 МБ, дальше из кэша).
Перетащи на страницу любое фото из `demo-photos/` или свое.

## Как это работает

1. CLIP кодирует загруженное фото в вектор (512 чисел).
2. Сравниваем его с заранее посчитанными векторами товаров (косинусная близость).
3. Параллельно по тем же векторам определяем категорию товара на фото.
4. Показываем топ-12 похожих, с фильтрами по категории и тегам и текстовым
   уточнением запроса.

## Ограничения прототипа

- До ~1000 товаров работает комфортно. Больше — нужна векторная база.
- CLIP точнее на фото товаров со светлым фоном.
- Первая загрузка модели ~50 МБ (потом кэш браузера).
